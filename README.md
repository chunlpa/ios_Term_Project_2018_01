* 추가적으로 수업에 배웠던것을 활용하여
* 1.알림예약을 추가해보았습니다.

* 2. 스톱워치를 추가하였습니다. 

* 3. 스토리보드로 피커뷰를 추가하여 이미지를 넘기려고 하였는데 오류를 알수없어서 추가하지 못했습니다.

* 4. 얼럿트를 사용하여 버튼을 입력시 전화걸기를 할수있게 만들었습니다
